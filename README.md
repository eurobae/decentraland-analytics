## Retrieving LAND Transactions from OpenSea
[OpenSea API](https://docs.opensea.io/reference/api-overview) is used to retrieve all the historical events (e.g., auctions, sales, transfers) of LANDs from 2018 to 2020. The retrieval is in progress; 75,560 transactions of 30,000 LANDs have been collected so far.
